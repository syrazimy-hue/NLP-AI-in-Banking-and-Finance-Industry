# NLP-AI-in-Banking-and-Finance-Industry
This repository contains the dataset and R analysis script used for an NLP-based study of artificial intelligence in the banking and finance industry.
The repository is prepared to support transparency, reproducibility, and verification of the text-analysis workflow used in the study.

## Repository Contents
```text

```

## Dataset

The dataset is stored in:

```text
data/dataset_clean
```

The dataset contains cleaned textual data used as input for the NLP-based analysis. The data were prepared for text mining, sentiment analysis, topic modeling, or other related natural language processing procedures used in the study.
If the dataset contains abstracts, titles, or publication-level metadata obtained from bibliographic databases or publisher platforms, users should ensure that reuse complies with the relevant data-source terms and conditions.

## Analysis Script

The main R analysis script is stored in:
```text
data/analysis_main_script
```
The script contains the R code used to conduct the NLP-based analysis. It may include procedures such as data loading, text preprocessing, sentiment analysis, emotion analysis, topic modeling, visualization, and result generation.

## Software Requirements

The analysis was conducted using R and RStudio.
Please check the `analysis_main_script` file for the exact packages used in the analysis.

## How to Reproduce the Analysis

1. Download or clone this repository.
2. Open the repository folder in RStudio.
3. Open the analysis script:

```text
data/analysis_main_script
```

4. Install the required R packages if they are not already installed.
5. Run the script from top to bottom.
6. The script will use the cleaned dataset stored in:

```text
data/dataset_clean
```
